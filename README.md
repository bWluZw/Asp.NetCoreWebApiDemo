# Asp.NetCoreWebApiDemo
基于Asp.NetCore的WebApi项目demo
已完成或已添加功能：
JWT鉴权
异常捕获（中间件方式）
日志（log4）
多数据库适配和原生sql扩展方法执行（目前仅支持MySql、SQL Server、Oracle）
动态WebApi（Swagger，UseKnife4UI）
自动依赖注入（基于AutoFac）
动态Entity实体注册（基于AutoFac）
对象数据映射（AutoMapper）
cors基本配置
json相关（NewtonsoftJson）
实现基本仓储模式（后续会慢慢整合）

未完成：
